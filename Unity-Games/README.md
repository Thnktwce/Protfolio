# Unity / Games / Prototypes

This folder contains mini-games and prototypes developed in Unity.  
Each project demonstrates core game mechanics, physics, AI, UI, and other aspects of game development.

## Projects

- [Graffiti Jump](https://github.com/Thnktwce/graffitijump) - Platformer prototype with custom movement and obstacles.
- [Tetris Core Mechanics](https://github.com/Thnktwce/Tetris-Core-Gameplay) - Core mechanics implementation of Tetris.
- [FPS Unity Mechanics](https://github.com/Thnktwce/FPS) - Basic FPS movement, shooting, and enemy AI.
- [Memory Game](https://github.com/Thnktwce/Memory-Game) - Simple memory match game built in Unity.
